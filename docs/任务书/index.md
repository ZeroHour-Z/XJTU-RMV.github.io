# 任务书

任务书是 2025 年西安交通大学 RoboMaster 笃行战队视觉组的任务说明以及指引。

任务书主要包括任务说明/任务指引/思考题（不定期更新，并非必要任务，不在任务提交中回答，但是进行思考或许可以对读者的理解有帮助）。
